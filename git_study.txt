git 工作区
git 暂存区 stage/index
git 版本库 .git文件

git add file	从工作区--> 暂存区

git commit -m '注释'  从暂存区-->版本库

git commit -am '注释' 从工作区-->版本库

查看 提交历史记录 
git log
git log --preety=oneline 一行显示

查看 版本历史记录
git reflog

git status 查看暂存区状态

git branch 查看分区
git checkout -b 分区名  创建分区
git branch -d 删除分区
git branch -D 强制删除分区

git remote -v 查看git服务器  

HEAD 代表最新版本库 
HEAD^ 上个版本库
HEAD^^ 上上个版本库
HEAD　~[0-N] 第几个版本库
用 git reflog 命令查看

git checkout -- 文件名    撤销工作区的修改，从版本库获得
git reset HEAD readme.txt 撤销暂存区的修改
场景1：当你改乱了工作区某个文件的内容，想直接丢弃?工作区的修改时，用命令
git checkout -- file
场景2：当你不但改乱了工作区某个?文件的内容，还添加到了暂存区时，想丢弃修改，分两
步，第一步用命令
git reset HEAD file，就回到了场景1，第二步按场景1操作。


rm file 从工作区删除
从版本库删除文件
git rm file 
git commit
